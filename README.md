# User Stories

## Description

As a User, I want to utilize the CS540 Student Application to search student information by ID number.

## Acceptance Criteria

### AC 1
The application should run using python3.
### AC 2
The application should provide input prompt.
### AC 3
The application should print student name based on the input.
### AC 4
The application should print error message if the input id is invalid or not found.


## Assumption (User, Technical, or other Business assucmptions)

### User
User should be able to run python application on command line.

### Technical
- Developer should have a basic knowledge of Python programming language.
- Developer should have a tool to update the input text file.

## Limitations and Constraints
- Access to the GIT & GITHUB is a must.
